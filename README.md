Personal Expense Tracker

A full-stack Personal Expense Tracker Web Application built using Django that helps users manage daily expenses, track spending patterns, and visualize financial data through an interactive dashboard.

This project demonstrates practical implementation of backend development, database integration, authentication systems, and frontend data visualization.

Project Overview

The Personal Expense Tracker allows users to securely log in and manage their expenses in an organized way. Each expense entry is stored in a database and displayed on a dynamic dashboard with graphical analysis.

The project is built using the Django framework (maintained by the Django Software Foundation) and version-controlled using GitHub.

Features

User Authentication (Login & Logout)

Secure session-based access to dashboard

Add and manage daily expenses

Categorized expense tracking

Interactive dashboard with charts (Chart.js)

Responsive UI design

SQLite database integration
