**💸 SpendWise — Personal Expense Tracker**

A full-stack web application built with **Python & Django** that helps users
track their daily income and expenses, visualize spending patterns, and manage
their personal finances — all behind a secure login system.

**🚀 Features**

- 🔐 Secure signup, login & logout (Django session-based auth)
- ➕ Add income & expense transactions with title, amount, category, and date
- 🗑️ Delete transactions with Post/Redirect/Get pattern (no duplicate submissions)
- 📊 Live dashboard — net balance, total income, total expenses
- 📈 Chart.js line chart — income vs expense trends over time
- 🗂️ Category-wise breakdown — Food, Transport, Health, Bills, and more
- 👤 Per-user data isolation — users only see their own transactions
- ⚙️ Django Admin panel — manage all users and expenses at /admin

**🛠️ Tech Stack**

| Layer        | Technology                     |
|--------------|--------------------------------|
| Backend      | Python, Django                 |
| Database     | SQLite (Django ORM)            |
| Frontend     | HTML, CSS, JavaScript          |
| Charts       | Chart.js                       |
| Auth         | Django built-in auth system    |
| Version Control | Git & GitHub                |
